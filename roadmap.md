# Obsidian Tag Navigator - 开发路线图

## 项目概述

Obsidian Tag Navigator 是一个为 Obsidian 笔记应用设计的插件，旨在为页面根据标签（tags）和 frontmatter 属性提供智能的下一页（next）和上一页（prev）导航功能。该插件通过分析笔记的元数据，为用户创建基于语义和分类的浏览体验。

## 核心功能

### 🎯 主要特性

- **基于标签的导航**: 根据笔记的标签创建逻辑导航序列
- **智能排序**: 支持多种排序方式（时间、标题、自定义顺序等）
- **跨文件夹导航**: 不受文件夹结构限制的语义导航
- **标签管理页面**: 独立的标签排序和自定义管理界面
- **Obsidian标准设置**: 集成到Obsidian插件设置面板

### 🔧 技术实现

- 基于 Obsidian API 开发
- TypeScript 实现，提供类型安全
- 响应式设计，支持桌面和移动端
- 模块化架构，代码分离清晰
- JSON 存储自定义排序配置

## 技术实现细节

### 核心组件

1. **TagNavigatorPlugin** (`main.ts`)
   - 插件主入口
   - 命令注册和视图管理
   - 插件生命周期管理
   - 设置管理

2. **工具函数模块** (`src/utils/`)
   - **NoteUtils** (`noteUtils.ts`): 笔记获取、标签处理
   - **SortUtils** (`sortUtils.ts`): 排序逻辑和排序描述
   - **ExportUtils** (`exportUtils.ts`): 标签导出功能
   - **NavigationUtils** (`navigationUtils.ts`): 前进后退导航

3. **视图组件** (`src/views/`)
   - **NavigatorPanelView** (`NavigatorPanelView.ts`): 侧边面板界面
   - **ManualNavigationModal** (`ManualNavigationModal.ts`): 手动排序界面
   - **SettingsPageView** (`SettingsPageView.ts`): 标签管理页面
   - **TagNavigatorSettingTab** (`SettingsTab.ts`): Obsidian标准设置面板

### 用户界面特性

- **分离式设置系统**: 主要设置在Obsidian设置面板，标签管理在独立页面
- **智能标签排序**: 三种排序模式满足不同使用需求
- **自动保存**: 拖拽排序后立即保存，无需手动操作
- **视觉反馈**: 完整的拖拽动画和状态指示器
- **响应式设计**: 支持桌面和移动端设备

### 数据流

1. **笔记扫描** → 提取标签和元数据
2. **排序处理** → 根据设置应用排序规则
3. **JSON 存储** → 自定义排序保存在插件设置中
4. **导航执行** → 根据排序结果进行导航

### 存储格式

**插件设置 JSON:**
```json
{
  "sortOrder": "title",
  "showToastMessages": true,
  "customOrder": {
    "research": ["note1.md", "note2.md", "note3.md"],
    "project": ["project-start.md", "project-end.md"]
  },
  "exportFolderPath": "tag-exports",
  "exportFileNameFormat": "tag-{tag}.md"
}
```

**导出的标签笔记格式:**
```markdown
# Tag: #research

## Notes List

1. [[Introduction to Research]]
2. [[Research Methodology]]
3. [[Data Collection]]
4. [[Analysis Framework]]
5. [[Conclusions]]

---
> *This note was automatically generated by Tag Navigator plugin*
> 
> Generated on: 2024-01-15 10:30:00
> Sort Order: Custom Order
> Total Notes: 5
> 
```
